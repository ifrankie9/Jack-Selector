File|New Clear inputs. Create a new Jack selector.

File|Open Open an existing selection.

File|Save Save the current selection.

File|Save As Save the current selection under a new name, and make the newly named selection the current image.

File|Print Print all detailed information about the current selection.

File|Quit Terminate the program. You will be given the chance to save any unsaved changes.

If there are numbered files, these are recently opened files. Choosing one of them will cause it to be opened and to become the current selector.
